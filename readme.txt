Stage 1: 
1. git config --global user.name "Your Name"
2. git config --global user.email youremail@domain.com
3. git clone (url), instead of the (url), 
		type the address of the GitHub repository 
		that you copied from your GitHub page; 
		(e.g. git clone https://github.com/isalirezag/Test.git).

Stage 2: 
1. git init
2. git add adding_file_name(like README.md)
3. git commit -m "first commit"
4. git remote add origin repo_path(like git@github.com:BDNapster/paymentdue_onepage_html.git)
5. git pull origin master
6. git push -u origin master
